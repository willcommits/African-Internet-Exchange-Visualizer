Introduction

Our project provides a specific online tool for academics to observe and simulate Africa's Internet connectivity via IXPs and AS.The platform features three distinct views:

1. IXP examine: Displays African IXPs based on their specific geolocation, allowing users to pick an IXP, examine all linked AS, and access important information such as IXP ID, name, and net count.

2. National View: Displays the AS connections of each African country. When a nation is selected, all of its AS and their relationships are displayed, with colors denoting customer-to-provider (c2p), peer-to-peer (p2p), and sibling-to-sibling (s2s) linkages.

3. Edit Studio: Allows researchers to import and view network data from CSV or Json files, broadening the area of visualization beyond simply African networks. Users can also change, add, or remove nodes in this area.

The 'Experiment' function in the first two views allows researchers to model prospective changes in Internet infrastructures by updating, adding, or deleting nodes and connections.

User Manual

Once the application is successfully running in your browser, you'll have the capability to perform the following tasks:

1. Adding a Node: IDs are auto-generated; users fill a form detailing node type, name, website, relations, city, and status.

2. Adding a Link: Select a source node, click 'Add Source', choose a destination, click 'Add Destination', then 'Add Link' to connect. Manual ID input is disabled for data integrity.

3. Removing a Node: Users can effortlessly remove a node by simply clicking on the node they wish to delete, followed by clicking the 'Delete' button. This action will remove the selected node from the system

4. Search IXP on Map: Users can find an IXP by name or longname, and the map auto-navigates to its location, displaying its network connections.

5. Search Country: In the national view, users can search for a country to display its associated AS networks, aiding map navigation and exploration.
